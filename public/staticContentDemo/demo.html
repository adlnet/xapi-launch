This is a demo of using static content with the XAPI Launch platform. When you click the button, your LRS will include the statement.

This is a link to another page on another domain. Since the attribute "courseLink" is set to "true" the launch library will transport your launch token to this content. Note that since the page you are on now has already exchanged the token for a session cookie from the client, the next page in the course must also use client side tracking.
